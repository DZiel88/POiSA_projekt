1. Program działa wyłącznie na plikach .wav.
2. Zbyt duży rząd filtrów Czebyszewa i Butterwortha może przeciążyć program obliczeniowo.
3. Wielokrotne używanie przycisku "Zastosuj efekty" będzie stosować efekty na uprzednio zmodyfikowanym pliku. Efekty będą się nawarstwiały.
4. Do korzystania z programu należy zainstalować wykorzystywane w projekcie biblioteki.
